# Python OSINT script tools
For those who seek to learn about Open Source Inteligence (OSINT) through a simplified lens, those scripts provide a simple and easy to understand syntax for some of the most widely used techniques

**port_scanner.py**  
Searches for open ports on a specified address 

**subdomain_scanner.py**  
Searches for subdomains on a specified domain, user must provide a file containing a list of subdomains for search

**username_scanner_search.py**  
Searches for a given username throught various social media, user may specify sites than the ones provided
